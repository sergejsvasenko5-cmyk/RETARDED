# Expense Tracker - Трекер расходов

## Автор
**Имя и фамилия:** Священко Сергей Владимирович

## Описание программы
Expense Tracker - это десктопное приложение с графическим интерфейсом для отслеживания личных финансов. Программа позволяет добавлять расходы, фильтровать их по категориям и датам, а также подсчитывать общую сумму расходов за выбранный период. Все данные сохраняются в JSON-файл.

### Основные возможности:
- Добавление расходов с указанием суммы, категории и даты
- Автоматическая проверка корректности ввода (сумма > 0, дата в формате ГГГГ-ММ-ДД)
- Фильтрация расходов по категории и диапазону дат
- Подсчёт общей суммы расходов за указанный период
- Сохранение и загрузка данных в формате JSON
- Удобный табличный интерфейс на основе tkinter
